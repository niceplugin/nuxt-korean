이 디렉토리에는 일반적으로 다음과 같은 유형의 파일이 포함됩니다:

- 스타일시트(CSS, SASS 등)
- 폰트
- [`public/`](/docs/guide/directory-structure/public) 디렉토리에서 제공되지 않는 이미지

에셋을 서버에서 제공하고 싶다면, [`public/`](/docs/guide/directory-structure/public) 디렉토리를 참고하는 것을 권장합니다.
